关于 vibe coding 最先问的问题之一：*要花多少钱？*

2025 年 7 月，SaaStr 创始人 Jason Lemkin 发了一篇文章，标题是《我这个月在 Replit 上可能会花掉 8000 美元，我觉得没问题》。他在 3.5 天内就烧掉了 607 美元，而他以为每个月只要 25 美元。这不是个例：2025 年中 Cursor 更改计费模式后，有用户单日超额账单超过 7000 美元；一位开发者追踪了自己 8 个月的 Claude Code 使用记录，按 API 单价换算相当于消耗了 1.5 万美元以上。很多硅谷程序员人依靠大公司平台，每天$200-300很正常，偶然可能一天超过1千美刀 。我调研过某国内开发为主的初创公司，节省一些，也需要每月$1000+。 即使只是一千美金，对于在车库里成长的startup，还是比较多的。

好消息是：今日大部分token是被浪费掉的。Anthropic 自己公布的数据显示，Claude Code 用户平均每天花约 6 美元，90% 的用户每天不超过 12 美元（月均约 360 美元上限）。那些骇人听闻的账单来自浪费，而不是工作本身。行业研究发现，编程 AI 代理所用 token 中有 70% 是浪费的——上下文爆炸、任务拆分不清、长会话中失去焦点。简单的人工介入，就是可以节省一半，甚至90%的token浪费。通过模块化设计、明确 interface、把任务拆小，对于 Claude Code 来说，就是一个任务拆分成10 个小而聚焦的模块来完成，而不是一道复杂的哥德巴赫猜想。需要理解的上下文减少了，大大降低token的消费。账单也跟着缩小。

![](/blog/assets/B3_token_waste_modular.svg)

**真实情况：比你从未去过的健身房会员便宜**， 当然这也是检验你是否是合格的软件架构师和产品经理。

以下是构建和运行 FIRE51 的完整成本拆解——一款包含 iOS/Android 壳、JWT 认证、MySQL、税务引擎和 PolicyEngine 验证套件的生产 Web 应用。

## 技术栈

| 项目 | 成本 | 说明 |
|------|------|------|
| **域名**（IONOS） | $10–15/年 | fire51.com — IONOS 的 .com 定价明显低于 GoDaddy/Namecheap |
| **AWS Lightsail** | $12/月 | 2 GB 内存 · 2 vCPU · 60 GB SSD — 支撑开发、测试、预发布和生产 |
| **Cursor** | $0 | 免费版 — 用作带 AI 补全的主 IDE |
| **Claude Code** | $20/月 | Pro 计划 — 主要 AI 编程代理 |
| **ChatGPT** | $20/月 | GPT-4o — 用于研究、起草、对架构进行第二意见 |
| **硬件** | 沉没成本 | MacBook Pro M1 2020，8 GB 内存——没有 GPU，没有服务器，只是一台笔记本 |

**每月总计：约 $53/月**（含域名约 $636/年）

![](/blog/assets/B3_cost_pyramid_donut.svg)

## $53/月 能得到什么

- 一个上线于真实域名、带 SSL 的生产 Web 应用
- 同一台服务器上的预发布环境（独立的 PM2 进程）
- 全天 24 小时可用的 AI 结对程序员，永不疲倦，从不抱怨代码库
- 两个 AI 系统用于交叉验证：Claude 用于实现，ChatGPT 用于研究和审查
- 带内联补全的完整 IDE（Cursor）

对比：一位高级开发者20分钟的咨询费用。

## 钱究竟花在哪里

$12/月的 Lightsail 实例在做的事比看起来多：
- PM2 管理两个进程（生产 + 预发布/开发）
- MySQL 服务应用、存储报告、OTP 代码、税务验证样本
- Nginx 为两者处理 SSL 终止和反向代理
- Node.js 运行 Express 服务器 + 模拟引擎

2020年购买的 8GB 内存的 M1 MacBook 已经过了它的当打之年，老骥伏枥任然足够应对本项目要求。——TypeScript 编译、本地 MySQL、运行测试、带 AI 补全的 Cursor——但它不是强力机器。同时打开 Chrome、IDE 和本地开发服务器时偶尔会有些卡顿。考虑到大部分计算机的寿命在~5–8 years，这台6年新基本算是45岁程序员焕发青春了。

## AI 工具分工

**Claude Code（$20/月）：** 主力工具。所有代码生成、重构、调试、文档写作和测试编写。代理循环——读取 → 规划 → 实现 → 测试——是大部分 vibe coding 发生的地方。

**ChatGPT（$20/月）：** 研究和第二意见。税法边缘情况、IRS 出版物查找、架构权衡。它是研究助手，Claude Code 是实现者。

**Cursor（免费）：** IDE 层。内联补全和快速文件导航。用于代理工作的频率低于 Claude Code，但在 Claude Code 处理另一个文件时有助于在代码库中保持方向感。

## 入门Vibe Coding 的人意味着什么

不需要强大的机器、昂贵的 IDE 许可，或复杂的云设施。门槛是：

1. $20/月 用于一个有能力的 AI 代理（Claude Code 或同类产品）
2. $12/月 用于一台小型云服务器（或从免费 VPS 开始）
3. 一台能运行代码编辑器和终端的笔记本

其他一切——框架、运行时、数据库、Web 服务器——都是开源且免费的。

Vibe coding 的真实成本是**注意力，而不是金钱**。需要阅读每一个 diff、验证每一个设计决策，并在整个过程中保持质量判断。AI 负责打字，产品经理和架构师负责思考。这就是这笔交易，而且是一笔好买卖。

## 补充说明

**¹ ChatGPT 在这里不是额外成本。** $20/月的订阅早在这个项目之前数年就已经订阅了。如果只需要选一个 AI vibe coding 工具，仅 Claude Code 就能覆盖大部分的需求。

**² 你可以以 $5/月 运行这一切。** 选择 $12/月 的 Lightsail 实例是为了支持通过 Puppeteer 进行的服务端 PDF 生成，这是个内存消耗大的功能。跳过这个功能——或使用 `window.print()`——$5/月 的套餐（512 MB 内存，2 vCPU，20 GB SSD）完全可以胜任 Node.js + MySQL + Nginx。从那里开始，只有在遇到真实约束时才升级。

**³ 关于 $53/月 的视角。** 一家典型的软件公司为单个贡献者支付的token成本是 $1,000–$3,000+/月，还不保证他们会交付东西。这套技术栈用其中的一小部分就能交付一个生产级全栈应用。

**⁴ 服务页面不消耗任何 token。** 这个博客在构建时就被生成为静态 HTML。当读者访问时，服务器只是发送预构建好的文件——运行时不会发生任何 AI API 调用。所有的 token 成本都发生在创作阶段：写文章、翻译、构建。一旦发布，无论多少流量，都不会产生额外的 AI 成本，只有固定的 $12/月 服务器费用。这是一个经过刻意设计的架构：AI 在你的电脑上完成工作，而不是在你的数据中心里。

**⁵ 掌控自己的基础设施。** 2025 年 7 月，Replit 删除了 Jason Lemkin 的生产数据库——1206 条精心整理的高管记录——Replit 将这次事故的严重程度自评为 95/100。这就是在托管平台上做 vibe coding 的风险：你不掌控服务器，就不掌控数据。$12/月 的 Lightsail 实例是你自己的：有 SSH 访问权限，可以做完整备份，你和你的数据之间只隔着你自己的决定。看起来更便宜的平台，往往是更贵的故障。

**⁶ Claude Code plan的选用。** 我使用的是入门版$20/月。一方面是因为这个项目不需要大规模Agent或程序员团队合作，程序逻辑也比较简单，可以使用默认的Sonnet 4.6，而不是高端的Opus4.6。另一方面是我个人只计划用三分之一的时间vibe coding，如果5小时的窗口提前用完，就去遛猫、踢球、做饭和读书，劳逸结合，而不是赶进度。如果是全职coding的话，还是建议$100的计划。

写在最后，本篇不是为优化工程人力成本的公司写的教程。它是为那些在车库里构建梦想的人写的。
